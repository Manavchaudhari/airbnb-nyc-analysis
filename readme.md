# Airbnb Listings Analysis

data analysis of New York City Airbnb listings using Python, pandas, seaborn, and SQLite for SQL queries. 
---

## Dataset

- **Source:** Inside Airbnb – New York City detailed listings
- **File used:** `listings.csv` → renamed to `data/raw/listings_raw.csv`
- **Key fields used:** price, room_type, neighbourhood, minimum_nights, availability_365, number_of_reviews, host_id, calculated_host_listings_count, latitude, longitude

---
## Technologies

Python: pandas, numpy, matplotlib, seaborn

SQL: SQLite

## How to Run

Create and activate the virtual environment:

python -m venv .venv
source .venv/bin/activate (linux/mac)  OR .venv\Scripts\activate (windows)


## Install dependencies:

pip install -r requirements.txt
